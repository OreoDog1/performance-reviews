Performance Reviews


1. Make a copy of this document by clicking File -> Make a Copy and give the spreadsheet any name you would like.

2. Fill in the data by inserting the name and team of each reviewer and reviewee into their respective sheets.

3. Optionally, you may include data under the "Experienced" column in the "Reviewers" sheet. This allows you to ensure that every reviewee will have an experienced reviewer. If you do this, the requested data points are "y" and "n". Otherwise, please delete this column to avoid confusion.

4. Now download both sheets by clicking File -> Download -> Comma Separated Values (.csv). Make sure to download both the "Reviewers" and "Reviewees" sheets individually by selecting one at a time and repeating the process for each.

5. Finally, input the downloaded files as well as the number of reviewers you would like to review each reviewee into the following form. Click "Submit" and you will be able to view the assignments and download them as a csv.
